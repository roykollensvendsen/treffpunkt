# Treffpunkt

Record and score your shots on official Norwegian Shooting Federation
(NSF / ISSF) targets, then compare results across competitions.

- **Users:** start with [Getting started](user/getting-started.md).
- **Developers:** start with the [Developer overview](dev/index.md) and
  [Architecture](dev/architecture.md).

Treffpunkt is in early development; see the [Roadmap](ROADMAP.md).
